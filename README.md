## ✨ Features

- Responsive Design — works on mobile, tablet, and desktop
- Smooth Scroll Navigation with active link highlighting
- Animated Skill Bars triggered on scroll (IntersectionObserver)
- Project Cards with staggered fade-in reveal
- Contact Form with client-side validation and user feedback
- Scroll-to-Top floating button
- Mobile Hamburger Menu with full-screen overlay
- Clean Typography — DM Serif Display + DM Sans

## 🌍 Deploy on Netlify (Recommended)

1. Go to [netlify.com](https://netlify.com) and sign up free
2. Click **"Add new site" → "Deploy manually"**
3. Drag and drop the portfolio folder
4. Get a live URL instantly — share it everywhere!

## 🌍 Deploy on GitHub Pages

1. Push code to a GitHub repository
2. Go to **Settings → Pages**
3. Set Source to `main` branch → `/ (root)` → Save
4. Live at ``

## 🔧 How to Update

| What to Change | Where in index.html |
|----------------|---------------------|
| Profile links | Hero section social icons |
| About text | About section paragraphs |
| Skill levels | `data-level` attributes on skill-bar divs |
| Project details | Projects section article cards |
| Contact details | Contact section info items |
| Accent color | `style.css` → `:root` → `--accent` variable |
